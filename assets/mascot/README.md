# CoBot mascot assets

CoBot is CoAct's robot-astronaut mascot.

| File | Intended use |
|---|---|
| `icon.png` | CLI/package avatar and small README placement |
| `mascot.png` | Product pages, launch posts, and larger documentation art |
| `sprite-sheet.png` | Motion, status-state, and future terminal/UI experiments |

The current CLI does not print bitmap art in the terminal. Keeping the mascot
in documentation avoids terminal compatibility and accessibility problems.
